# Turn-Based Combat System Portfolio

Unity 기반 턴제 전투 시스템 개인 프로젝트입니다.

## Tech Stack
- Unity
- C#
- Git

---

## Implemented Features

### Grid Movement System
- Grid 기반 이동 시스템 구현
- BFS 기반 Pathfinding 적용

### Turn-Based Combat
- AP(Action Point) 기반 턴 시스템
- 연속 행동 처리
- 행동 취소 흐름 구현

### Skill System
- 스킬 사용 구조 구현
- 타겟 선택 및 범위 처리

### Status Effect System
- Burn / Poison / Stun 상태이상 구현
- StatusEffect 기반 확장 구조 설계

### AI System
- Utility 기반 AI 의사결정 구현
- 상황별 행동 우선순위 계산

### VFX Integration
- 스킬 및 상태이상 이펙트 연동

### Gameplay Flow
- 턴 진행 흐름 관리
- 행동 취소 시 상태 복원 처리
