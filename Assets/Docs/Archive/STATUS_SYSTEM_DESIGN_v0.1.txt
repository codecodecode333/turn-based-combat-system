============================================================
1. Status System Overview
============================================================

StatusEffect는 유닛에게 지속적으로 영향을 주는 상태 효과이다.

상태는 다음 정보를 가진다.

type
power
duration
source

type
상태 종류

power
상태 효과 강도

duration
지속 턴 수

source
상태를 적용한 유닛


============================================================
2. Status Lifecycle
============================================================

Status는 다음 흐름으로 처리된다.

Apply Status
→ Turn Start Tick
→ Duration 감소
→ Duration 0 이면 제거


============================================================
3. Status Tick Timing
============================================================

Status 효과는 턴 시작에 발동한다.

Trigger

OnTurnStart


예

Burn

턴 시작 시
Burn damage 적용


============================================================
4. Status Stacking Policy
============================================================

Stack 정책

StrongestOnly + DurationMax


동일한 Status가 이미 존재할 경우


Case 1

새 power > 기존 power

→ 새 Status로 교체


Case 2

새 power < 기존 power

→ 기존 power 유지
→ duration = max(existing, new)


Case 3

새 power == 기존 power

→ duration 갱신
→ duration = max(existing, new)


예

Existing

Burn
power = 5
duration = 2


New

Burn
power = 3
duration = 5


Result

Burn
power = 5
duration = 5

Shield
피해 흡수


Buff

Debuff

6. Status Limit
============================================================

한 유닛이 가질 수 있는 Status 수

Unlimited


단, 동일 타입 Status는

Stack 정책을 따른다.


============================================================
7. Status Removal
============================================================

턴 시작 시

duration -= 1


duration <= 0

→ Status 제거
